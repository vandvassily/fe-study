# Sentry

Sentry是一个开源的监控系统，能支持服务端和客户端的监控，同时还有强大的后台错误分析和报警平台。官方网站：

Sentry.ioopen in new window

# sentry能力介绍

  1. JS代码错误诊断
  2. 记录页面加载性能
  3. 追踪用户轨迹
  4. API请求监控

# 使用

配置为只有生产环境启用 sentry。
